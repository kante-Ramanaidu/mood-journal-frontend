


# Mood Journal App

A full-stack web application to track, record, and reflect on your daily moods with personalized quotes and music suggestions.

---

## Features

- User authentication: Sign up and login securely
- Mood selection with emoji icons and descriptive tags
- Display of motivational quotes and mood-based music recommendations
- Mood history tracking for self-reflection
- Responsive design for mobile and desktop

---

## Technologies Used

### Frontend

- React.js
- CSS (custom styling)
- Axios for API requests

### Backend

- Node.js with Express.js
- PostgreSQL database
- JWT authentication
- External APIs integration (e.g., YouTube API for music)
